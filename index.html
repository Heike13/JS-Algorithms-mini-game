Mini Jeu : Rejoint la case rouge

Tapes les commandes la dedans :

Le but du jeu est de déplacer le curseur noir de la cellule bleue à la cellue rouge

Pour cela, tu peux te déplacer en tapant ces commandes dans le tableau noir ci dessus

  • Pour faire tourner le curseur rouge vers la gauche, tape : gauche
  • Pour faire tourner le curseur rouge vers la droite, tape : droite
  • Pour faire avancer le curseur rouge vers la case pointée, tape : avance
  • Commande spéciale : de avanceX2 => jusqu'à => avanceX6

Tu peux relancer une nouvelle partie s'il n'y a pas de chemin jusqu'à l'arrivée, le développeur n'a pas encore chercher de solution
